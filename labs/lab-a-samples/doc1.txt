Security Advisory SA-2026-001: A critical authentication bypass vulnerability has been identified in ExampleApp versions 3.x prior to 3.4.2, allowing unauthenticated remote code execution via a crafted HTTP request to the admin API endpoint. Organizations running affected versions should apply the vendor patch immediately and restrict network access to the admin interface pending remediation. Temporary mitigation: block inbound requests to /admin/api/ at the perimeter firewall until the patch is deployed.
