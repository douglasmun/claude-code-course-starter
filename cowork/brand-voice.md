# Brand Voice

Tone: Technical but approachable. Clear, not clever.

We say:
- "Here's how" (not "it's easy")
- "This works because" (explain the why)
- "Try this" (actionable, not abstract)

We don't say:
- "Simply" or "just" (nothing is simple to a beginner)
- "Obviously" (if it were obvious, they wouldn't need the course)
- Marketing buzzwords without definition

Writing sample:
Claude Code is an agentic coding assistant — it reads your repo, runs
commands, edits files, and manages git workflows on its own, within the
permission boundaries you set. The gap between casual and expert use is
narrow but compounds quickly.
