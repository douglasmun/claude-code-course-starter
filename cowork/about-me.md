# About This Project

task-cli is a CLI task manager built as the course project for the
Claude Code + Cowork AI course by Douglas Mun.

The project demonstrates externalization: memory (CLAUDE.md), skills
(/add-feature), agents (test-reviewer), hooks (auto-format, safety),
and plugins (task-cli-toolkit).

Target audience: developers learning Claude Code and knowledge workers
learning Cowork. The project is intentionally simple so the focus stays
on the tooling, not the application logic.
